# [CP] Resources Generators

## Description

大型可制造器械, 自动化生产基础物料: 泥土, 纤维, 石头, 木头等.

## 机器种类

### 泥土机

1. 100% 泥土
2. 005% 煤炭
3. 003% 晶石
4. 001% 万象晶石

### 纤维机

1. 100% 纤维
2. 005% 混合种子

### 石头机

1. 100% 石头
2. 005% 铜矿石
3. 004% 铁矿石
4. 003% 黄金矿石
5. 002% 铱矿石

### 木头机

1. 100% 木头
2. 010% 硬木

## 商店更改
